# 美语跟读陪练 — 使用说明

一个网页工具：粘贴英文 → 逐句美音朗读 → 跟读练习。已发布在线，直接打开网址即用：

**https://victorliu86.github.io/english-shadowing/**

## 手机上怎么用（一次设置，永久一键）

1. 华为应用市场（AppGallery）安装 **Microsoft Edge** 浏览器
2. 在 Edge 里打开上面的网址（可以先把网址发到微信文件传输助手，复制粘贴）
3. Edge 菜单 → **添加到主屏幕**，桌面上就有一个像 App 一样的图标了

> 需要联网（语音是在线合成的），但不依赖 Google 服务。本地文件 `index.html` 与网页内容相同，电脑上可直接双击使用。

## 声音质量分两档（自动切换，看标题旁的小徽章）

| 徽章显示 | 什么时候 | 效果 |
|---|---|---|
| **微软神经语音** | 用 **Edge 浏览器**打开时 | 8 种可选声音（4 女 4 男，都是美音），非常接近真人（强烈推荐）|
| **标准语音** | 用华为自带浏览器等其他浏览器打开时 | 2 种可选：女声（美音）/ 男声（英音），清晰标准，但机感稍重 |

> 注意：兜底档的男声是**英音**（免费接口里没有男声美音）。想听男声美音（Andrew/Guy 等），必须用 Edge 浏览器打开。

**推荐做法**：在华为应用市场（AppGallery）搜索安装 **Microsoft Edge** 浏览器，用它打开本页面，即可解锁微软神经语音（Ava / Andrew 最自然）。电脑上 Windows 自带的 Edge 打开也一样有神经语音。

注：微软的接口只认 Edge 浏览器，且要求 Edge 版本较新——如果切不过去，先把 Edge 升级到最新版。

## 功能

- **▶ 从头朗读**：全文逐句连读，当前句高亮
- **点任意句子**：单独听那一句（精听神器）
- **⏮ / 🔁 / ⏭**：上一句、重读本句、下一句（底部常驻栏）
- **跟读模式**：勾选后，每句读完会静音等待同样的时长，让你大声复述
- **语速**：慢 → 快共 5 档（变速不变调）
- **⬇ MP3**：把全文合成一个 MP3 下载，存手机里通勤路上反复听（仅微软语音模式可用）

## 推荐的跟读（Shadowing）练法

1. **盲听**：先不看文字，点 ▶ 整体听一遍
2. **精听**：逐句点，听不清的句子反复点 🔁
3. **跟读**：开「跟读模式」，每句后大声模仿它的语音语调
4. **对比**：手机录下自己的朗读，和原声对比差在哪

## 出问题时

- 显示「已切换到标准美音」：说明微软语音连不上（不是 Edge 浏览器，或 Edge 版本太旧，或微软改了接口）
- 显示「已切换本机语音」：在线语音都连不上，检查网络
- 微软偶尔会改接口导致神经语音失效；标准美音兜底一般不受影响。彻底失效时找 Claude 更新这个文件即可
